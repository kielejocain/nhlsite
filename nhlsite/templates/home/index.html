Welcome!

This site is my attempt to bring machine learning algorithms to bear on prediction of NHL fantasy stats. If you'd like more details about the method I'm attempting to employ, see the about:method page. If you'd like to know what features are coming down the pike, see the about:site page.

Where to look

If you have a favorite team, use the "Conference" dropdown lists above. If you're curious about a particular player, enter all or part of their last name in the search box above (note: I only have players that played at least one season since 1997-1998, and only predict for players that played last season; the model has nothing to say about how Gretzky would fair in today's NHL). If you'd like to know more about the specifics of the model, check the "Stats" dropdown to see features of the algorithms for specific stats (as I make them). If you know nothing at all about hockey, take a gander at the glossary page to learn about basic hockey definitions, then come back here and try one of the players from the 2015 All-Star rosters to see an example of a good player's numbers.

Thanks for visiting!
